## Задание 3
Выведите в консоль три строки. Первая – обычная строка. Вторая – F строка с  использованием заранее объявленной переменной. Третья – сложите две или более строк в  одну.

  ```python
print("Привет, мир!")

hello = "мир!"
print(f"Привет, {hello}")

hi = "Привет, "
world = "мир!"
print(hi + world)
```

## Краткий вывод:
Для вывода обычного текста необходимо использовать "" в принт - print("Привет, мир!")

F строка необходима для более удобного сочетания текста и переменных вычисляемых в процессе выполнения кода - print(f"Привет, {hello}"), где hello это переменная

Несколько текстовых переменных можно сочитать между собой при помощи знака +  - print(hi + world), где hi и world это переменные.

## Задание 6
Создайте две любые числовые переменные и выполните над ними несколько математических  операций: возведение в степень, обычное деление, целочисленное деление, нахождение  остатка от деления. При желании вы можете проверить как работают эти вычисления с  разными типами данных, например, сначала создать две переменные int, затем создать две  переменные float и наконец создать переменные типа int и float и провести над ними  операции, прописанные выше.

  ```python
one = 47
two = 7
print("Возведение в степень: ", one ** two)
print("Обычное деление: ", one / two)
print("Целочисленное деление: ", one // two)
print("Нахождение остатка: ", one % two)
```

Нахождение остатка производится с помощью %
